## 안녕하세요, 백엔드 개발자 김범승입니다.

Java와 Spring Boot를 중심으로 서비스 흐름을 설계하고 구현합니다.<br>
최근에는 AI 기능이 들어간 서비스에서 백엔드, Python API, 외부 API가 안정적으로 이어지도록 만드는 일에 관심을 두고 있습니다.

---

### 👋 저는 이런 개발을 좋아합니다

- 사용자가 만나는 기능 뒤에서 데이터가 어떤 순서로 흐르는지 끝까지 확인합니다.
- API 계약, 콜백 처리, 예외 상황처럼 서비스가 흔들리기 쉬운 지점을 먼저 정리하려고 합니다.
- 팀 프로젝트에서는 작업을 작게 나누고, 기록과 리뷰가 가능한 형태로 남기는 방식을 선호합니다.
- 단순히 기능을 붙이는 것보다 운영 흐름과 문제 해결 과정을 함께 설계하는 백엔드를 지향합니다.
